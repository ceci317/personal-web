# Ceci Personal Web (English)

This cyberpunk-inspired portfolio site shows Ceci's AI product mindset, growth experiments, internship recaps, and personal storytelling.

## Highlights

- Neon-green + black palette with animated lighting and grid background
- Single-page flow covering About → Quote → Projects → Job Experience → Contact
- Project section pairs narrative copy with looping video previews and direct links
- Contact row offers icon-only entry points plus a WeChat/phone modal for quick adds
- Extra standalone pages (`ai-builder.html`, `becoming.html`, `beyond-ai.html`, `contact.html`) for deeper storytelling

## Tech Stack

- HTML / CSS / Vanilla JS
- Three.js canvas hero background

## Files

- `index.html`: primary landing page with hero, about, quote, projects, and contact
- `style.css` + `script.js`: bespoke styling and interactivity
- `assets/`: videos, imagery, QR code, resume, and related media
- `scripts/`: helper scripts for semantic analysis and structured outlines
